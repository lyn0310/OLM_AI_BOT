#### 개요
· 목적
  사내 생산하는 모든 제품은 표준원가가 생산 전 계산되어야 한다.
  계약사육(총액)의 경우는 표준원가를 마크, 릴리즈 하지 않으나 BOM,라우팅, 생산버전 정보는 사전 등록이 되어야 한다.
· 내용
  신규 제품이 생성되었거나 기존 제품이나 반제품의 조달 유형이 변경된 경우 기준정보 체크리스트를 수행하여 BOM누락, 라우팅 누락, 표준원가 누락을 확인하여 생성하도록 한다.
· 시스템 구현방안
  기준정보 점검 리포트를 개발하여 에러 발생내용을 확인 가능하게 한다.
  생산계획이 정상될 경우 당월 생산계획 기준으로 에러발생 내용을 담당자에게 리포팅 되게 한다.
