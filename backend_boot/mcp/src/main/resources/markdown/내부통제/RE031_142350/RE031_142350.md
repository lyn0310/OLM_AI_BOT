#### 개요
1. 고객지원팀 비용담당자는 월 마감 후 익월 초에 모두싸인 시스템으로 각 거래처별로 [거래사실 확인원]을 매월 전자서명으로 발송한다.

 등기로 발송되며, 발송하였던 등기번호를 관리하고 있다.
