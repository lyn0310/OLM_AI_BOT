### 상세 속성 (Attributes)
#### 개요
- 하림그룹의 표준 생산방식을 생산오더 방식과 반복제조 방식의 두가지 방법으로 각각 표준화하여,
 - 각사의 BU별 특성을 고려하여 생산오더 방식과 반복제조 방식을 결정하여 사용할 수 있도록 함

#### 만족도
- ,

#### 기대효과
- 표준 프로세스의 도입으로 신규 프로세스 및 변경에 유연하게 대처가 가능함
 - 시스템과 제조현장 실적의 동기화로 물/정 일치를 실현하여 전사 관점의 데이터 분석이 가능함
 - 계획대로 생산하는 체계를 구축하여 제조 현장 관리 수준의 고도화 가능

#### 비고/이슈
50.03.03.01 사내 작업 실적 처리
50.03.03.02 생산오더 부산물 실적 처리
50.03.03.03 예외 작업 실적 처리
50.03.04.01 반복제조 정상 실적 처리
50.03.04.02 반복제조 예외 실적 처리
 SAP-PP / MES

#### 전제조건 및 제약사항
- 각 사별/업무 영역(BU) 생산방식이 우선 결정되어야 함 (1. 생산오더 방식, 2. 반복제조 방식)
 - 주별/일별 생산계획을 수립하여 최소 1일 전 또는 당일 생산 시작 전에는 작업지시가 생성되어야 함

#### TOBE 변화사항
- - 하림그룹의 각 사/BU 별 생산방식 (생산오더 생산, 반복제조 생산) 을 표준화하여, 생산현장의 Data 를 표준화된 기준으로 처리하고, 처리된 Data 를 ERP 에 반영하여 생산 관련 물/정 Data 를 일치화하여 생산관련 Data 의 신뢰성을 향상
