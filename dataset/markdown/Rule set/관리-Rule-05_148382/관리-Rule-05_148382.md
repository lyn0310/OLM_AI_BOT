### 상세 속성 (Attributes)
#### 개요
목적
- 물동흐름(Movement Type)을 원가수불 관점에서 집계하여 누락된 항목이 존재하는 지 여부 검증
- 제조결산 전 수행하여 관리(CO)에 통보되지 않았거나 정상적이지 않은 거래 식별


내용
- SAP 표준 제공 자재원장(Material Ledger) 기반으로 해당 물동 흐름을 식별하고 원가수불 관점에서 검토하여 
  누락되었거나 정상적이지 않은 항목이 존재하는 지를 CBO 리포트를 통하여 식별
- 해당 건에 대해서는 월결산 시작전에 확인하고 오류 건으로 판명 시 정상처리 하도록 가이드하고
  재발하지 않도록 업무협의 진행


시스템 구현방안
- 제조원가 결산 사전 점검 리포트의 하나로 CBO 프로그램 개발


연계 시스템
- ERP 內 SD/MM/PP와 연계하여 검증
