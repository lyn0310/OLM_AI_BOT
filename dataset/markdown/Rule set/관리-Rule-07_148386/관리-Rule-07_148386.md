### 상세 속성 (Attributes)
#### 개요
목적
- 플랜트 단위 원가결산 마감의 정합성 제고 측면에서 한 법인의 발생 제조비용을 다른 법인으로 배부하는 경우 원가결산마감 불가


내용
- 하나의 관리회계영역(Controlling Area) 내 복수 개의 회사(Company Code)가 존재하나,
  제조원가 배부 시 법인(회사코드) 간 배부가 이루어지지 않도록 업무적으로 가이드하여 배부 사이클(주기) 구성
- 배부 사이클(주기)를 정례적으로 점검하여 해당 위험이 존재하는 지 사전에 점검


시스템 구현방안
- 배부 사이클(주기)를 점검하는 CBO 프로그램 개발 및 월 결산 시작 전 사전 검증

#### 비고/이슈
- 회사/법인(Legal Entity)간에 관리회계(CO) 관점에서 배부가 이루어지면 안됨

#### 적용방안상세
원가중심점 회계(Cost Center Accounting, 이하 'CCA') 모듈을 이용한 제조원가 배부 시 법인(회사코드) 간 제조비용 배부 금지
- 동일 회사(법인) 코드 내 CCA 사이클 생성이 이루어지도록 제조원가 배부 사이클 구성
