### 상세 속성 (Attributes)
#### 개요
· 목적
  ALT BOM과 라우팅 그룹이 복수 일 경우 생산진행 시 해당하는 생산버전이 생성되어야 한다.
· 내용
  생산대상인 제품/반제품을 기준으로 ALT BOM과 복수 라우팅 그룹의 조합으로 생산버전이 생성되었는지 확인한다.
· 시스템 구현방안
  생산버전 일괄 조회 프로그램에서 미생성된 라우팅 및 BOM을 조회하고 미사용 상태처리를 하거나 추가 생성하도록 한다.
  생산버전 누락을 확인한다. (사내/계약(순액))에 대해서도 생산프로세스를 사용할 경우 대상으로 한다.
· 연계 시스템

 - 사내는 BOM + 라우팅 조합으로 관리
 - 위탁생산은 저장위치별 생산버전 관리
2. 상세구현방안
 - 신규 제품이 생성되었거나 조달 유형이 변경된 경우 생산버전 관리(사내제품/사내반제품)
 - 외주 공정을 사내외주처럼 관리할 경우도 생산버전 관리
 - 외주 가공(Subcontracting)에 대해서도 생산버전 관리
 - Phantom 자재에 대해서도 생산버전 관리
