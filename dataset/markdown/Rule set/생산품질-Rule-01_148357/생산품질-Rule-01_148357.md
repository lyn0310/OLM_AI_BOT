#### 개요
· 목적
  : 생산현장 시스템인 MES 생산 실적 Data 와 ERP 시스템인 SAP 의 생산실적 Data 의 일치를 위해 지켜야 하는 Rule
· 내용
  : MES 는 SAP 에서 생성된 Production Order 에 의해서만 작업지시를 처리한다
    SAP 의 Production Order에 대한 생산 실적은 MES로 부터 수신하여 처리하고 실적 취소도 MES에서 수신한 데이터를 근거로 처리한다.
· 시스템 구현방안
  : 1) SAP 에서 생산오더르 생성하는 기능과 MES 로 전송하는 기능 구현
    2) MES 에서는 SAP 의 생산오더를 수신하는 기능 구현
    3) SAP 의 생산오더 번호를 기준으로 처리된 MES 의 생산실적 Data 를 MES 에서 SAP 로 전송하는 기능 구현
    4) MES 의 생산실적 Data 를 SAP 에서 수신하여 생산실적 처리를 하는 기능 구현
       ( MES 기준으로 생산출고, 생산공수, 생산입고 처리하는 Function 기능 포함)
· 연계 시스템
  : 1) 각 사에서 정의된 생산현장 시스템 (MES)
