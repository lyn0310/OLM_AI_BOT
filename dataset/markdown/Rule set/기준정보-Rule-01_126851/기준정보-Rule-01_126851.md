#### 개요
. 목적 : 품목코드/BP데 대한 그룹 Governance 유지 관리

. 내용 : 품목코드/BP의 생성/변경/이관/삭제는 ERP에서 통합 거버닝하여 배포한다.


. 시스템 구현합안
   - 품목코드/BP의 생성, 변경, 단종(폐기) 는 Groupware를 통한 승인 완료된 건에 대해서만 ERP에서 처리함
   - ERP에서 수불이 발생하지 않는, Legacy에서만 필요한 품목코드라도 ERP에서 통합 관리 후 필요한 Legacy System에 Interface 함
   - ERP 등록 시 Validation 점검을 통과한 대상만 정상 처리되게 함
   - 신청은 각 업무 담당자가 수행하며 Data의 정합성 및 ERP 처리는 통합기준정보팀에서 수행함


. 해당 Rule Set이 적용되는 조건
   - 품목유형 전체, BP 전체에 대하여 적용하며 생성, 변경, 단종 처리 시 적용

. 연계시스템 : Groupware
