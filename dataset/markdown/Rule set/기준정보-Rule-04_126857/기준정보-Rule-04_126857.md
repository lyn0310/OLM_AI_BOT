. 시스템 적용방안
 - 품목코드 신규생성, 모듈 관련 기준정보 준비 상태, 단종 등의 품목코드 Life Cycle 관리를 위하여 자재상태(Material Status) 기능을 사용함 - 그룹 1물 1품목코드 운영으로 Org. Level의 자재상태 기능 적용 (CLient Level이 아님) . 동일 품목코드가 한 조직에서는 수불이 발생하지만 다른 조직에서는 단종이 되는 경우가 있음 ㆍ해당 Rule Set이 적용되는 조건 - 품목코드 신규 생성시는 'NW(New) 값으로만 생성되게 함  - BP 코드는 General, Company Code, Purchasing/Sales Org. Level 로 사용 가부를 관리 함(Blocking/Relaease Status) - FI(회계) 에서 추가로 요청 하는 법적인 상태도 추가로 Status 관리 함 ㆍ해당 Rule Set이 적용되는 조건 
 생성 시 Blocking 을 걸고 생성 요청자가 필드 등을 최종 확인 후 Release 하여 사용. 파산, 법적조치 등 회계와 관련 된 상태 관리도 함께 함
